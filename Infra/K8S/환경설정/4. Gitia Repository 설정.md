# Gitia Repository 설정

## 회원가입
`http://<Gitia IP>:30480/` 경로로 접속
**Register**에서 운영 계정을 만든 뒤 로그인한다. 우측 상단의 `+` 메뉴에서 **New Repository**를 선택하고 Helm Chart를 저장할 Repository 이름과 공개 범위를 지정한다.

Repository 생성 후 표시되는 Clone URL을 사용한다.

```shell
git clone <Git 주소>
cd <Repository 이름>
```

폐쇄망에서는 DNS와 인증서 신뢰 설정을 먼저 확인하고, 개인 계정보다 배포 전용 계정 또는 Deploy Key를 사용한다.
### 저장소 설정 (우리는 helm으로 배포할 예정이라 아래와 같다.)
```shell title="아래와 같이 배포할 저장소 init"
[root@exscms1 ieee160921]# ls
Chart.yaml  templates  values.yaml
```
### values.yaml 파일 수정
repository IP 설정
```yaml title="values.yaml"
image:
  repository: "10.19.2.171:5000/ieee160921-scms"
  tag: "latest"
```
