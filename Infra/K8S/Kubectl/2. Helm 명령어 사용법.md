# Helm 명령어 사용법
## Helm Chart 이용하여 Deployment 실행
```shell title="Helm Deployment 실행"
helm install <deployment-name> <values.yaml-file-root-path> -f <my-values.yaml> -n <name-space> --create-namespace

#argo-cd를 설치하는 예시
helm install argo-cd . -n argocd --create-namespace

#argo-cd-image-updater를 설치하는 예시
helm install argocd-image-updater . -f my-values.yaml -n argocd --create-namespace
```
## Helm으로 설치한 Deployment 삭제
```shell title=""
helm uninstall <deployment-name> -n <namespace>
```
