# VIM 잊어먹기 쉬운 커맨드
## 커맨드
- **r** : 문자 1개 변경
- **%s/바꿔질문자/바꿀문자/g** : 해당 문자 모두 변경
- **f** : 커서 이후의 문자 찾기
- **F** : 커서 이전의 문자 찾기
- **ctrl + a** : 숫자 1 증가
- **ctrl + x**/ : 숫자 1 감소
- **{** : 커서 이전의 공백이 있는 라인으로 이동
- **}** : 커서 이후의 공백이 있는 라인으로 이동
- **;** : 이전 명령 다시 실행
- **,** : 이전 명령 롤백
- **q** : 매크로
- **@q** : 매크로 실행
- **@숫자q** : 매크로 숫자만큼 반복 실행
